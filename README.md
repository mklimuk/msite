# msite

> South America consulting website based on Jekyll

## To get started

```sh
$ gulp
```

And you'll have a new Jekyll site generated for you and displayed in your
browser. Neato.

## Usage

```sh
$ gulp build
```

```sh
$ gulp publish
```

```sh
$ gulp deploy
```
